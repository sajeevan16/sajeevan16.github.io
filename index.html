PacGuard (API Security)

Reinforcement Learning based system to detect and mitigate API attacks.

Role

Researcher & Developer. Conducted experiments on DDoS patterns and trained the RL agent.

Tech Stack
Python Keras / TensorFlow Reinforcement Learning (DQN) Docker Flask API
Architecture & Design

The system models API security as a game where the Agent (PacGuard) learns to block malicious requests while allowing legitimate traffic. It uses a Deep Q-Network (DQN) to predict the optimal action (Allow/Block) based on request features (rate, size, source). The environment is simulated using Docker containers representing legitimate users and attackers targeting a Flask-based API endpoint.

Nano Processor Design

Design and simulation of a 4-bit microprocessor.

Role

Hardware Designer. Implemented the ALU and Control Unit logic.

Tech Stack
VHDL Logisim Assembly Language
Architecture & Design

The processor features a custom Instruction Set Architecture (ISA) capable of basic arithmetic and logic operations. It includes a 4-bit Arithmetic Logic Unit (ALU), a Program Counter, Instruction Register, and a Control Unit that decodes opcodes into control signals. The design was verified through rigorous timing simulations in Logisim.
